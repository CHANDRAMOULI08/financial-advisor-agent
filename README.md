# 💰 Financial Advisor Agent

A personal financial advisor agent built on Base44. Provides general money advice, tips, and tracks financial queries.

## Features
- General money advice & tips
- Covers: Budgeting, Savings, Investing, Debt, Insurance, Taxes
- Tracks all financial queries in a database
- Prioritized financial action plan (The Stack)

## Financial Priority Order (The Stack)
1. Build $1,000 starter emergency fund
2. Get full employer 401(k) match
3. Pay off high-interest debt (>7% APR)
4. Build 3–6 month emergency fund
5. Max out Roth IRA ($7,000/year)
6. Max out 401(k) ($23,000/year)
7. Invest in taxable brokerage / real estate

## Skills
- `financial_advisor.py` — Core advice engine covering all financial topics

## Usage
Ask questions like:
- "How should I start investing with $500/month?"
- "I have credit card debt, what should I do?"
- "Give me budgeting tips"
- "What is the 50/30/20 rule?"

## Tech Stack
- Built on [Base44](https://base44.com) Superagent platform
- Python skill for advice generation
- Entity-based query tracking

## Disclaimer
This agent provides general financial advice and education — not personalized legal, tax, or investment advice.
